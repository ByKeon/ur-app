1. 確認資格

  • • 年滿 18 歲
  • • 持有支援 NFC 晶片掃描的護照
  • • 居住在 UR 服務支援的國家 (查詢連結在下面)
  • • 非美國人
↓

2. 註冊 UR 帳戶

  • • 使用 Google 或 Email 註冊
  • • ✔ 輸入正確的驗證碼，驗証成功即可建立帳號
↓

3. 完成 KYC

  • • 填寫問卷
  • • GPS 定位
  • • 護照 NFC 掃描
  • • 人臉驗證
↓

4. 等待審核

  • • 通常 5 分鐘至幾小時，最長 3 個工作日
  • • ✔ 完成 KYC 審核後即可獲得瑞士銀行帳戶
↓

5. 啟動帳號（領取 Debit Card 卡號）

  • • 存入至少等值 5 美金的 USDC
  • • ✔ 可取得 Debit Card 卡號
